How to Use VulnEye – User Guide

VulnEye is a web-based vulnerability scanning dashboard designed to help users simulate and track security scans on different types of inputs such as URLs, files, images, and videos. This guide explains each section of the application and how to use it effectively.

1. Login & Access

  • Users must register or log in to access the scanner.
  • After successful login, the dashboard opens automatically.
  • Session is maintained using secure session storage.

2. Dashboard Overview

The Overview page gives a quick summary of your scanning activity.

  • Total Scans – Total number of scans performed.
  • Critical Findings – Number of high-severity issues detected.
  • Safe Scans – Scans where no vulnerabilities were found.
  • Recent Scan Activity – Displays your latest scans with severity labels.

Purpose: Helps users quickly understand their security status at a glance.

3. Scan Modules

This section allows you to perform different types of scans.

  • URL Scanner – Enter a website URL and click Start Scan.
  • Image Scanner – Upload image files (PNG, JPG).
  • Document Scanner – Upload PDF, DOC, or TXT files.
  • Video Scanner – Upload video files for analysis.

Each scan result is automatically saved to the database.

4. Scan History

  • View scan type, target, severity, and time.
  • Filter scans by severity or scan type.
  • Search scans using keywords.
  • Download individual scan results.
  • Permanently delete scans when needed.

5. Reports

  • Generate reports in JSON, CSV, Text, or PDF format.
  • Print all scans directly from the dashboard.

6. Settings

  • Theme Selection – Switch between Light and Dark mode.
  • Account Update – Change username or password securely.

7. About Section

The About section provides guidance and explains how to use every feature of VulnEye. It helps first-time users understand the workflow and functionality of the system.

8. Logout & Security

  • Use the Logout button to securely end your session.
  • Session data is cleared and the user is redirected to the login page.

Summary: VulnEye offers a simple, structured interface to simulate vulnerability scanning, manage scan records, and generate reports.
